Parking

2023

Outils utilisés

  • Php
  • Laravel
  • MySQL
  • GitHub
  • Ubuntu

L'application web, conçue à l'aide du framework Laravel, a pour but de gérer les places de parking de la M2L. Son objectif est de permettre à un employé de la M2L de réaliser plusieurs actions : réserver une place de parking, consulter son historique de réservations et être ajouté sur une liste d'attente si toutes les places du parking sont occupées.
